# Zachary Berardo Personal Site

Static personal-brand site for berardo.dev.

## Local Preview

```sh
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Files

- `index.html` contains the full one-page site.
- `styles.css` contains responsive layout, color tokens, and motion states.
- `script.js` contains section reveal, active navigation, mobile menu, and card spotlight behavior.
- `assets/zachary-berardo-headshot.jpg` is the optimized headshot.
